hexo博客搭建

Windows 常用 cmd 指令

windows 下在地址栏输入 cmd 直接打开 command 地址为当前

linux —–> windows
pwd chdir
ls dir
cd cd

第一步

安装 node.js
网址——> node.js.org
Windows 建议下载 .msi 格式，下载完成无脑 “下一步” 0-0
顺便下载 git 并安装

第二步

安装并更换 npm 源
npm install -g cnpm registry=https://registry.npm.taobao.org (马爸爸牛逼0-0)
回车，等

cnpm -v 回车看版本
cnpm install -g hexo-cli 回车,等
hexo -v 回车看版本

第三步

建立一个 blog 文件夹
cd 到 blog
hexo init 直接初始化一个博客 （安装 git 后可以顺利进行） 等

第四步

输入 hexo s
localhost 4000 打开阅览器 输入 localhost：4000 进入
Ctrl + c 断开

第五步

hexo n “myblogname”
cd —> source/_posts
查看生成的博客
使用编辑软件打开就可以进行编辑
编辑完保存退出

第六步

cd 到 blog 目录处
cd ../.. 返回两层目录
hexo generated
