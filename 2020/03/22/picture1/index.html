如何将图片上传到Hexo博客上呢

2020-03-22
字数统计: 390字   |   阅读时长≈ 1分钟

图片导入博客

环境安装

  1. 将主页配置文件_config.yml 里的 post_asset_folder: 这个选项设置为true
  2. 在 hexo 目录下执行 npm install hexo-asset-image –save
    这是下载安装一个可以上传本地图片的插件
  3. 等待一小段时间后，再运行hexo n “xxxx”来生成md博文时，/source/_posts 文件夹内除了xxxx.md文件还有一个同名的文件夹

导入图片

  1. 在xxxx.md中想引入图片时，先把图片复制到xxxx这个文件夹中，然后只需要在xxxx.md中按照markdown的格式引入图片：![你想输入的替代文字](xxxx/图片名.jpg)

  2. 为什么有的人无法输出图片呢，会报错？

    • 没有在 blog (就是你运行 hexo 的地址) 下安装图片插件 ————》 重新安装
    • 检查你的插件版本，使用 npm install https://github.com/CodeFalling/hexo-asset-image –save 安装0.0.5版本的hexo-asset-image插件。 使用 npm install hexo-asset-image –save 安装的是1.0.0版本的hexo-asset-image插件。 两者最直接的区别是映射关系不同。 具体详情见 http://www.99708.com.cn/2019/07/26/hexo-本地图片不显示 ——————》重新安装就可
